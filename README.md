# ML-Electricity-price-forecasting
The aim of this project was to forecast electricity prices in the European day-ahead market for the Germany / Luxembourg bidding zone. To predict this machine learning models are trained and evaluated on a dataset spanning from 2018 to 2025 and containing a number of meteorological, economic, grid-specific and calendar features.

You can find the data cleaning and modelling code in the price_forecasting_v2 folder, the main results and a description of the project can be found in DSF_Electricity_Price_Forecasting_Antonin_Ricard
